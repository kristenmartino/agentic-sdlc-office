# Roadmap

## v0.1 — Visual Workflow Prototype

Visual office, placeholder agents, scripted demo. No real runtime.

## v0.2 — Real Agent Simulation

Agents run from prompts, produce artifacts, hand off work between rooms.

## v0.3 — GitHub Integration

Read/write GitHub Issues, PRs, and Projects from the office UI.

## v1.0 — Portfolio-Ready Demo

Polished visuals, narrated scenarios, deployable demo build.

## How this maps to milestones

| Milestone | What "done" looks like |
| --- | --- |
| `v0.1 Visual Workflow Prototype` | Clickable office, scripted scenario plays end-to-end |
| `v0.2 Agent Runtime Simulation` | Agents take real inputs and produce artifacts |
| `v0.3 GitHub Integration` | Issues/PRs round-trip from office UI |
| `v1.0 Portfolio Demo` | Deployable, narrated, public-facing |
