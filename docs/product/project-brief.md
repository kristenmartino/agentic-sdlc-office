# Project Brief

The one-pager. If a new contributor reads only this, they should know what we are building and why.

## Pitch

A visual operating model for the Agentic Development Lifecycle (ADLC). Agents work in a rendered office; humans govern from a Decision Inbox. The product makes invisible agent work *legible* — a portfolio piece that demonstrates how to design human-in-the-loop AI delivery.

## What it is

- A visual office with 8 agents and 8 rooms.
- A work-item stream that moves between agents and rooms.
- A Decision Inbox where humans approve, reject, or defer.
- An event log that makes every run replayable.

## What it is not

- A real autonomous coding agent (not yet).
- A general-purpose project management tool.
- A simulation of a particular company.

## Who it is for

- Hiring managers and design leaders evaluating Kristen's portfolio.
- Practitioners curious about ADLC-style operating models.
- Future me, planning the v0.2 runtime.

## Why now

ADLC reframes the SDLC from sequential phases to concurrent modes with agents executing under human governance. The industry has the *concept* but no good *visual model* of it. This project fills that gap.
